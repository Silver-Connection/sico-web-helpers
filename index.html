Transaction Model

The Transaction class is a set of tools for dealing with server response which match the TransactionModel interface.

The actual data is wrapped which make life with Vue.js better, because we can update our data as we like. 

interface TransactionModel {
    /**
     * Transaction action name
     * @default null
     * @type string
     */
    Action: string,

    /**
     * Transaction return code
     * @default null
     * @type string
     */
    Code: number,

    /**
     * Actual data
     * @default null
     * @type string
     */
    Data: any,

    /**
     * Transaction message
     * @default null
     * @type string
     */
    Message: string
}
